Steps to Open the Web

1) Required XAMPP Module with Apache and MySQL
2) Go to locahost/phpMyAdmin
3) Create Database Name "cateringdatabase"
4) Go to the database > import > file from .//databases/cateringDatabase (inside folder)
5) Go to http://localhost/webCateringGitHub/Pinocone-Catering-Software/login.php
6) Ready to Go!
